# 💸 Expense Tracker Dashboard – Power BI Project

## 🚀 Project Overview

The **Expense Tracker Dashboard** is a dynamic Power BI report built to visualize and monitor personal or household expenses. This dashboard helps users understand their spending habits across different categories, time periods, and payment methods.  
It supports better financial decision-making by offering clear insights into where the money goes.

## 📊 Dataset Description

The dataset is structured and manually logged in Excel format with the following fields:

1. **Date** – Date of the transaction  
2. **Category** – Type of expense (e.g., Food, Shopping, Car)  
3. **Description** – Short description of the item (e.g., McDonalds, Fuel)  
4. **Price** – Transaction amount in dollars  
5. **Credit Card** – Payment method used (e.g., Max, Discount, Pepper)

## 📈 Key Visuals in the Dashboard

1. 🟥 **Card – Total Spending**  
   → Displays total amount spent  
2. 🟦 **Card – Number of Transactions**  
   → Shows the total number of logged expenses  
3. 🟩 **Card – Average Expense Amount**  
   → Displays average amount per transaction  
4. 📉 **Line Chart – Daily Spending Trend**  
   → Tracks how expenses fluctuate day by day  
5. 🍩 **Donut Chart – Spending Distribution by Credit Card**  
   → Visualizes share of spending by payment method  
6. 📊 **Bar Chart – Top 5 Expense Items by Total Spending**  
   → Highlights the most expensive items or vendors  
7. 📊 **Column Chart – Monthly Average Expense**  
   → Shows spending habits by month  
8. 🍩 **Donut Chart – Spending by Category**  
   → Breaks down expenses across categories like Food, Shopping, etc.

---

## 🧩 Filters (Slicers)

- Date (timeline)  
- Category  
- Credit Card  
- Description  

These slicers allow users to drill down and explore specific time periods or expense types.
